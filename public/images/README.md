# Images Directory

This directory contains all image assets for the portfolio. Each subdirectory corresponds to a specific section or component.

## Directory Structure

### `/hero/`
**Purpose:** Hero section portrait image  
**Expected File:** `portrait.jpg` or `portrait.png`  
**Usage:** Main hero section on the homepage  
**Specifications:**
- Recommended size: 800x1200px (portrait orientation)
- Format: JPG or PNG
- Should be a professional portrait photo

---

## Naming Convention
- Use lowercase letters
- Separate words with hyphens (-)
- Use descriptive names that match the content
- Include file extension (.jpg, .png, .mp4)

## Notes
- All images should be optimized for web (compressed but maintain quality)
- Ensure images are properly licensed or owned
- Consider creating WebP versions for better performance
